# Installer **uv**:

**Sur macOS ou Linux**

Ouvre ton terminal et colle cette commande :

```bash
curl -LsSf https://astral.sh/uv/install.sh | sh

```

**Sur Windows**

Ouvre un terminal (PowerShell) et colle cette commande :

```powershell
powershell -c "irm https://astral.sh/uv/install.ps1 | iex"

```

# Initialisation

```bash
uv init
```
ou

```bash
uv init PATH
```
